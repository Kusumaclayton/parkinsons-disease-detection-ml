# 🧠 Parkinson’s Disease Detection using Machine Learning

This project presents a machine learning-based approach for early detection of Parkinson’s Disease using two key biomarkers: voice signal analysis and spiral drawing patterns.

The system leverages advanced algorithms such as XGBoost for voice data classification and Random Forest for image-based spiral analysis. By combining both modalities, the model improves prediction accuracy and provides a non-invasive, cost-effective solution for early diagnosis.

The project demonstrates the complete pipeline including data preprocessing, feature extraction, model training, evaluation using metrics like accuracy and confusion matrix, and final prediction.

🔬 Focus: Early diagnosis
⚙️ Tech: Python, OpenCV, Scikit-learn, XGBoost
📊 Approach: Multimodal Machine Learning System
